Internet Protocols

IP

Internet Protocol – addresses devices.

🌐 e.g., 192.168.1.1

TCP

Transmission Control – reliable, connection-oriented.

📦 e.g., Web browsing

UDP

User Datagram – fast, no connection.

🎥 e.g., Video streaming

HTTP

HyperText Transfer – web data.

🔗 e.g., Loading websites

HTTPS

HTTP Secure – encrypted.

🔒 e.g., Banking

FTP

File Transfer – upload/download files.

📁 e.g., Server uploads

SMTP

Simple Mail Transfer – email sending.

✉️ e.g., Gmail

DNS

Domain Name System – names to IPs.

🌍 e.g., google.com → 142.250.…
